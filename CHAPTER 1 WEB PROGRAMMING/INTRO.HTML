HTML & CSS Quick-Reference Guide

HTML Tags

<h1> - <h6>: Headings

Used to define headings, from largest (<h1>) to smallest (<h6>).

<h1>This is a Heading</h1>

<p>: Paragraph

Defines a paragraph of text.

<p>This is a paragraph.</p>

<a>: Anchor

Creates hyperlinks to other pages or resources.

<a href="https://example.com">Visit Example</a>

<img>: Image

Displays an image on a webpage.

<img src="image.jpg" alt="Description">

<ul>, <ol>, <li>: Lists

Defines unordered (<ul>) and ordered (<ol>) lists with list items (<li>).

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<table>: Table

Defines a table for displaying data.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
    </tr>
</table>

CSS Properties

color

Sets the color of text.

p {
    color: blue;
}

background-color

Sets the background color of an element.

body {
    background-color: lightgray;
}

font-size

Defines the size of the font.

h1 {
    font-size: 24px;
}

margin

Sets the margin around an element.

div {
    margin: 20px;
}
